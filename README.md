# Tambo
This is the tambola number generator I made.This is a project that is submitted to <a>https://horizons.hackclub.com/</a>

<h2 align="center">
  🌐 <a href="https://0xmanidev.github.io/Tambo/">Visit the Live Website</a>
</h2>


## Controls
- `d` draw a new number
- `r` reset the board
- `f` fullscreen

## Available themes
- Dark
- Light
- Catppuccin
- Nord
- Tokyo Night
- One Dark
- Horizons
- Hackclub
